# Salus Agenda Frontend

A visão geral do projeto está no [README da raiz](../README.md).

Se você entrou direto nesta pasta, este é o frontend da aplicação e os comandos principais são:

```bash
npm install
npm run dev
```

Para build de produção:

```bash
npm run build
```

Variável de ambiente usada pela app:

```env
VITE_API_URL=http://localhost:8080
```

Use a URL correta da sua API nesse valor.
